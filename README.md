# Emm...

该项目是Emm的Web管理界面，主要显示数据库ER图，为编码人员提供了解公司业务表结构的便捷方式，
帮助他们快速熟悉关键业务表，提高开发效率。


## 技术栈:

- Vite5
- React 18
- Jotai
- NextUi 2.2.9
- Tailwind CSS 3
- Sweetalert2
